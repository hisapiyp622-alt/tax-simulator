# 手取りシミュレーター

個人事業主(青色申告)向けの税金・手取りシミュレーター。売上と経費を入力すると、所得税・住民税(尼崎市)・個人事業税(兵庫県)を計算し、**年間の手取り額**と**税金納付に備えて毎月積み立てるべき金額**を表示します。

- iPhoneのホーム画面に追加して使うPWA
- データはlocalStorageに保存(サーバー不要)
- オフラインでも動作(service worker)

## 前提(利用者プロフィール)

- 兵庫県尼崎市在住・個人事業主(青色申告)
- 社会保険はソロ・コンシェルジュ経由で健康保険・厚生年金に加入(国保・国民年金ではない)→ 保険料は**月額固定入力**
- iDeCo 月23,000円・小規模企業共済 月1,000円がデフォルト
- 16歳未満の子2人(所得税の扶養控除対象外・住民税の非課税判定にのみ使用)

## ファイル構成

| ファイル | 役割 |
|---|---|
| index.html | 画面 |
| style.css | スタイル(白+深緑、積立カードは琥珀色) |
| app.js | UIロジック・localStorage |
| tax.js | 税計算(純粋関数)。**税率・料率の定数は冒頭の `TAX_CONSTANTS` に集約**。出典URLと確認日もここに記載 |
| manifest.json / sw.js / icon.svg | PWA関連 |

## 毎年の更新方法

1. `tax.js` の `TAX_CONSTANTS` を最新の税制に合わせて更新する
   - **特に基礎控除**: 合計所得132万超〜655万以下の上乗せ(88万/68万/63万)は**令和7・8年分限定**。令和9年分からはこの帯が58万円に下がる
2. `sw.js` の `CACHE_NAME` のバージョンを上げる(例: `tax-sim-v1` → `tax-sim-v2`)
3. `index.html` 内の `style.css?v=N` / `app.js?v=N` / `tax.js?v=N` の数字を上げる
4. コミットしてGitHub Pagesに反映されるのを待つ(1〜2分)

## GitHub Pagesへのデプロイ手順

1. GitHubでPublicリポジトリを作成(例: `tax-simulator`)
2. このフォルダの全ファイルをリポジトリ直下にpushする
3. リポジトリの **Settings → Pages** を開く
4. **Source: Deploy from a branch**、**Branch: main / (root)** を選んで Save
5. 数分後に `https://<ユーザー名>.github.io/tax-simulator/` で公開される
6. Actionsタブの「pages build and deployment」が緑になれば反映完了

## iPhoneのホーム画面に追加

1. SafariでアプリのURLを開く
2. 共有ボタン(□↑) → 「ホーム画面に追加」
3. 名前(手取り)を確認して追加 → 単独アプリとして起動できる

※アイコンはSVGを使用(GitHub Web UIから `icon.png`(180×180)を手動アップロードして `index.html` の `apple-touch-icon` を差し替えると、iOSでの表示品質が上がります)

## 免責事項

本アプリは概算のシミュレーションです。実際の税額・保険料は申告内容や自治体の決定により異なります。正確な金額は税務署・尼崎市・税理士にご確認ください。
